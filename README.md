# SnakeGame
This is a simple 2D Snake Game.

## Control keys
 - W -> Up
 - S -> Down
 - A -> Left
 - D -> Right

## Development
- Developed on Windows 11.
- Visual Studio 2022.

## Using tools
- C++.
- Lua.
- Premake (Link for Primake: https://premake.github.io/). 
- SFML (Link for SFML: https://www.sfml-dev.org/).
